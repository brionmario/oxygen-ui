## Overview

The SVG icons from `@oxygen-ui-experimental/primitives` exported as React components.

## Usage

Import and use the icons in your components as follows.

<Source
  language="jsx"
  dark
  format
  code={dedent`
import {LockIcon} from '@oxygen-ui-experimental/react-icons';\n
function Demo() {
  return <LockIcon size="small" />;
}`}
/>
